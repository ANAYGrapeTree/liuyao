# 六爻算命网站规范

## [S1] 项目初始化
- 使用Vite创建React TypeScript项目
- 配置Tailwind CSS
- 确保开发服务器可运行

## [S2] 问题选择界面
- 提供预设问题类别（情感运势、事业运势、财运、健康、学业）
- 支持自定义问题输入
- 用户选择问题后进入起卦步骤

## [S3] 起卦界面（铜钱摇卦法）
- 模拟铜钱抛掷，生成6个爻
- 支持老阳、老阴、少阳、少阴
- 动爻标记

## [S4] 排卦逻辑
- 实现六十四卦数据
- 装地支、六亲、六神、世应
- 显示排卦结果

## [S5] 解卦逻辑
- 基础解卦规则
- 用神分析、旺衰判断
- 显示解卦结果

## [S6] AI解读功能
- 支持用户输入MIMO API密钥
- 支持免费的默认LLM API
- 调用API生成解读

## [S7] 界面美化与响应式设计
- 现代简约风格
- 响应式布局，适配不同屏幕

## [S8] 部署到GitHub Pages
- 配置Vite构建
- 创建GitHub Actions工作流
- 自动部署到GitHub Pages

## [S9] 项目配置
- 添加.gitignore文件，排除node_modules、skills、缓存等非源代码文件
- 添加README.md文件，描述项目功能和使用方法
- 添加MIT LICENSE文件
