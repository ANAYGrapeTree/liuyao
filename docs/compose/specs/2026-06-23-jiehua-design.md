# 解卦逻辑设计文档

## [S1] 概述

解卦逻辑是六爻算命网站的分析模块，根据排卦结果进行分析解读，为用户提供占卜结果的解释和建议。

## [S2] 基础解卦规则

### 卦象解读
- 分析本卦和变卦的含义
- 解读卦辞和爻辞
- 分析卦象的整体趋势

### 爻的分析
- 分析每个爻的含义
- 考虑爻的旺衰状态
- 分析动爻的影响

## [S3] 用神分析

### 用神确定
- 根据问题类型确定用神
- 情感运势：妻财为用神
- 事业运势：官鬼为用神
- 财运：妻财为用神
- 健康：子孙为用神
- 学业：父母为用神

### 用神旺衰
- 分析用神的旺衰状态
- 考虑月建、日辰的影响
- 分析动爻对用神的影响

## [S4] 旺衰判断

### 月建影响
- 分析月建对各爻的影响
- 判断爻的旺衰状态

### 日辰影响
- 分析日辰对各爻的影响
- 判断爻的旺衰状态

### 动爻影响
- 分析动爻对其他爻的影响
- 判断动爻的吉凶

## [S5] 解卦结果

### 整体分析
- 卦象整体趋势
- 问题的总体走向
- 关键时间节点

### 具体建议
- 根据分析结果给出建议
- 注意事项
- 行动指南

## [S6] 界面显示

### 解卦结果显示区域
1. 整体分析
2. 用神分析
3. 旺衰判断
4. 具体建议
5. 注意事项

### 响应式设计
- 移动端：垂直排列
- 桌面端：水平排列

## [S7] 数据结构

```typescript
interface JieGuaAnalysis {
  overallAnalysis: string; // 整体分析
  yongShenAnalysis: string; // 用神分析
  wangShuaiAnalysis: string; // 旺衰判断
  suggestions: string[]; // 具体建议
  warnings: string[]; // 注意事项
}

interface JieGuaResult {
  analysis: JieGuaAnalysis;
  score: number; // 吉凶评分
  trend: '吉' | '凶' | '平'; // 整体趋势
}
```

## [S8] 组件结构

```
JieGua
├── OverallAnalysis (整体分析)
├── YongShenAnalysis (用神分析)
├── WangShuaiAnalysis (旺衰判断)
├── Suggestions (具体建议)
└── Warnings (注意事项)
```
