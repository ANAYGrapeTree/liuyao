# 六爻占卜网站

一个基于 React + TypeScript + Vite 的六爻占卜网站，提供在线占卜服务。

**在线访问：** https://anaygrapetree.github.io/liuyao/

## 功能

- 六爻占卜计算
- 现代化用户界面
- 响应式设计
- 快速构建和热重载

## 技术栈

- **前端框架**: React 19
- **类型系统**: TypeScript
- **构建工具**: Vite 8
- **样式**: Tailwind CSS 4
- **代码检查**: ESLint
